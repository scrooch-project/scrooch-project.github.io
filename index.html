🚀 What is Scrooch?

Scrooch is a modding framework built on top of popular Scratch mods. With Scrooch, you can quickly create your own customized Scratch mod without starting from scratch (pun intended 😉).

🛠 Features (Scrooch 3)

- Easy setup using GitHub templates
- Customizable UI and branding
- Works with extensions and custom blocks
- Based on proven modding engines for stability

💡 Why Choose Scrooch?

Scrooch saves you hours of setup time, so you can focus on adding features and designing your mod. Whether you’re creating a personal mod, a Mod for your friend, or the next big Scratch-based platform, Scrooch gives you the tools you need.
