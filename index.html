🚀 What is Scrooch?

Scrooch is a high-performance modding framework for building custom Scratch environments without starting from scratch.

🛠 Features (Scrooch 3)

GitHub Native
Quick setup using templates.
Custom Branding
Full control over UI and styling.
Mod Support
Works with custom blocks.
Addons
Extend mods using JavaScript plugins.

📦 Get Started

Choose your base and start building your mod.
