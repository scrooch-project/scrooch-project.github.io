🚀 What is Scrooch?

Scrooch is a high-performance modding framework. Build customized environments without the overhead of starting from zero.

🛠 Features (Scrooch 3)

✦ GitHub Native
Easy setup using templates.
✦ Custom Branding
Full control over UI and style.
✦ Mod Support
Works with custom blocks.
✦ Stability
Built on proven mod bases.
